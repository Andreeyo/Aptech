CASCADING STYLE SHEET

A Cascading Style Sheet (CSS) is a rule-based language, which specifies the formatting instructions for the content specified in an HTML page.
Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language. It was invented in 1997.

BENEFITS

Multiple HTML pages can use a CSS document.

  • Code Reusability
  • Less HTML Code
  • Device Independence.

    • CSS SYNTAX

    The general syntax of CSS consists of three parts namely, selector, property and value. A selector is an HTML element for which you want to specify the style or the formatting instruction. A property is a CSS property that specifies the type of the style to be applied to the selector. CSS allows controlling the appearance of the content by providing various properties. Which includes text properties, positioning properties, font properties, color properties etc. A value refers to the value of the CSS property. A CSS property can have multiple values. The property and its value for a selector are separated with a colon (:). They are enclosed within the curly brackets ({}) that is known as the declaration block.

    TYPES OF STYLE SHEETS

  • Internal style sheet is also included within the HTML document. It is defined using the style element within the HEAD element
  • External style sheet is defined in a separate file and is saved with the .css extension. It provides the benefit of reusability by implementing common style rules for multiple HTML pages. Linking the external style sheet to your web page you use: link rel=”stylesheet” type=”text/css” href=”body.css”
  • Inline CSS: here the styling is done inside a particular tag.

    • TYPES OF SELECTORS

    There are three(3) types of selectors namely: Type selector, Class selector, ID selector, Universal Selector.
    * Type Selector specifies the element name along with the styles to be applied to that element. This results in applying the specified styles to all the occurrence of that element in the web page.
    Type selector { color:brown; }
    * Class Selector matches elements, whose class attribute is set in an HTML page and applies styles to the content of all those elements.
    .class selector. { color:brown; }
    * ID Selector matches elements, whose ID attribute is set in an HTML page and applies styles to the content of all those elements.
    #id selector. { color:brown; }
    * Universal Selector represents all the elements with the HTML page. It matches all the elements in the HTML file. This means that it applies the specified styles to the content of all the elements.
    * Universal selector. { color:brown; }

    EVENT HANDLING

    Event can be defined as an occurrence of an action that can be notified by the source to the subscribers. Examples of events are mouse clicks, keystrokes etc.

    EVENTS DESCRIPTION
    onclick Specifies the action to be taken on mouse click.
    onload Specifies the action to be taken when a mouse pointer moves over an element.
    onmouseover Specifies the action to be taken when a mouse pointer moves over an element.
    onmouseout Specifies the action to be taken when a mouse pointer moves out of element.
    onsubmit Specifies the action to be taken when a form is submitted.
