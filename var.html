VARIABLES

Refers to a symbolic name that holds a value, which keeps changing. In JS, a variable is a unique location in the computer’s memory that stores a value and has a unique name. This name of the variable is used to access and read the value stored in it. A variable can store different data types.

DECLARING VARIABLES

Refers to creating a variable by specifying the variable name. In JS, the var keyword is used to create a variable by allocating memory to it. A keyword is a reserved word that holds a special meaning in JS.

SYNTAX:

Var variable_Name;
Where,
var:Is the keyword in JS.
variableName:is a valid variable name.

ASSIGNING VALUE TO VARIABLE

variableName = value;
Where
=: is the assignment operator used to assign values.
Value: Is the data that is to be stored in the variable.

DECLARING MULTIPLE VARIABLES

Declaring multiple variables in a single statement separated by commas.
var variableName1 = value1, variableName2 = value2;

RULES

You cannot refer to a variable until it is created in JS. JS is case-sensitive. In JS there are certain rules, which must be followed while specifying variable names. There are:

  • Can consist of digits, underscore and alphabets.
  • Must begin with a letter or the underscore character.
  • Cannot begin with a number and cannot contain any punctuation marks.
  • Cannot contain any kind of special characters such as +, *, % etc.
  • Cannot contain spaces.
  • Cannot be a JS keyword.

It is recommended to give meaningful names to a variable such that the name determines the kind of data stored in the variable.

HOW TO DISPLAY VARIABLES

Javascript allows display of info using methods of the document objects. This document object is a predefined object in Javascript which represents the HTML page and allows managing the HTML page dynamically.

The methods of displaying objects on browser with javascript are;

  • writeLn();
  • document.write();
  • alert();
  • confirm();
  • prompt();

writeLn():

Displays any type of data and appends (adds) a new line character.

alert():

Displays a dialogue box with some information and an ‘OK’ button

confirm():

Displays a dialogue box with the ‘OK’ and ‘CANCEL’ button. It verifies an action which the user wants to perform.

prompt():

Displays a dialogue box that accepts an input value through a textbox.
