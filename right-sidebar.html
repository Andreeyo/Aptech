DIV AND SPAN ELEMENTS

The DIV element is a block-level element that defines a section in a webpage. This means that it logically divides webpage into sections, which makes it easy to identify each section. The DIV element contains other block-level elements and inline elements. Both block-level and inline elements specify content to be displayed in the browser. However, block-level elements are preceded and followed by a blank line. Inline elements specify a piece of text and exist with the block-level elements. Examples of block-level elements include p and H1 elements. E.g. of inline elements include A and cite elements.

The SPAN element is an inline element that applies style to a part of the text. It is used in line with the text to specify the style. The two most commonly used attributes of the SPAN and DIV elements are id and class. The id attribute specifies a unique name to the element. The class attribute specifies a class name to which the element will belong. Multiple class names can be assigned to the element, which are separated by a space. The same name can be assigned to multiple elements.

USING LISTS

A list is a collection of items, which might be organized in a sequential or unsequential manner. List can be used to display related items that belong to a particular category. HTML allows you to display related items in a list on a web page. A list in HTML can contain paragraphs, line breaks, images, links and other list. This items within a list are displayed on a web page one below the other using bullets. HTML allows creating three types of lists namely, Ordered, unordered and definition.

ORDERED LISTS

Is a list of items arranged in a particular order. The order of the items is important as it indicates a sequential flow. HTML provides 2 elements for creating an ordered list and there are:
• OL: creates an ordered list.
• LI: specifies an item and it is a sub-element of the OL element.

UNORDERED LISTS

An unordered list is a list wherein the items are arranged in random order. This means that you will create an unordered list when the order of the related items is not important. HTML provides the UL element to create an unordered list.

“list-style-type” Property

The list-style-type property specifies the type of bullet to be applied to an unordered list. There are three types of bullets defined for the unordered list in HTML. These bullet types are disc, square and circle. This values are not case sensitive.

DEFINITION LISTS

A definition list refers to a collection of terms with their corresponding description. A definition list appears with the term indented on the left followed by the description on the right or on the next line. • DL: is a container element that consists of the DT and DD sub-elements.
   It specifies that a definition list will be created using these elements.
• DT: specifies the term to be defined or described.
• DD: specifies the definition or description of the term.

“IMG” ELEMENTS

The IMG element is an empty element, which inserts an image in a web page. It allows inserting pictures and diagrams. It supports the commonly used image formats such as GIF, JPEG, BMP and PNG.

OBJECT ELEMENT

The OBJECT element embeds an object in a web page. It allows including such as video files, audio files, flash files and web pages in a webpage. Apart from multimedia files, it also allows including other files such as .doc files, .xls files, .pdf files and so on. The element defines some attribute, which are used to specify the object, its type, height, width etc.
