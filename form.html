FORMS

An HTML form is a container that consists of various controls such as text boxes, buttons and labels. These controls allow you take information and view the requested information through the form. This means that HTML forms enhance user interactivity by providing the controls.

“Form” ELEMENT

You can construct a form by using the FORM element in HTML. The FORM element is a container element that allows you to create a form on a web page.


ATTRIBUTE DESCRIPTION
action Specifies the URL where the form data will be submitted and processed.
method Indicates an HTTP method that specifies how the data of the form is sent to the server.
This attribute has two values namely, GET and POST.
The GET method appends the form data to the URL specified in the action attribute.
The POST method sends the form data to the server as the request message without appending to the URL.
The value POST is used more often because it allows you to send large amount of data and ensure better security.

BASIC CONTROLS

A control is a visual object that you can add to an HTML form. It is used to accept input from user or to display information. Commonly used controls are;
• Label: Allows you specify a descriptive text, which a user cannot modify.
• Text Box: Allows you to input text in one line. In case of multiple lines of text, you can use the text area control. You can also specify the text to be displayed as dots. These dots form a password, which will prevent others from reading the text.
• Button: provides the easiest way to enable user interaction. The user can click the button to perform the required action. There are two buttons in HTML namely, SUBMIT and RESET. The SUBMIT button sends the form data to the server, while the RESET button resets the form entries to their default values.
• Check Box: Allows you to select one or more options from the specified list.
• Radio Button: Allows you to select a single option from a set of options.
