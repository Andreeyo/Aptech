What is a Browser?

A program used to view HTML documents.

What is a web?
The web is a way to access information using the internet that is referred to as a network of networks.
What is a webpage?
A webpage is a file that contains information and instructions to display the information to the users.
What is a web server?
Is a computer with high processing speed connected to the internet and is used to host webpages.
What is Internet?
A computer network consisting of a worldwide network of computer networks that use the TCP/IP network protocols to facilitate data transmission and exchange.

What is TCP/IP?
A set of protocols (including TCP) developed for the internet in the 1970s to get data from one network device to another. (Transmission Control Protocol/Internet Protocol).
What is a Dynamic webpage?
Generates content on “on- demand” when user provides certain inputs. It accepts the inputs from the user based on which it displays the content in the browser. E.g. online store. Dynamic websites can include webpages containing static as well as dynamic. Data can be stored and retrieved using the technologies used in creating dynamic websites. Some technologies used are: 1 JavaScript (JS). 2 Cascading Style Sheet (CSS). 3 Extensible HTML (XHTML). 4 Dynamic HTML (DHTML).

DEFINITION

HTML- HyperText Markup Language.

It is a markup language used to create and link webpages. A markup language defines a set of rules that specifies how the data is displayed on a browser. Was derived from Standard generalized Markup Language (SGML), which is the mother language of HTML. SGML is a markup language that defines the structure of the other markup languages.

ORIGIN OF HTML5

Was officially drafted in 2004 by WHATWG-WEB HYPERTEXT APPLICATION WORKING GROUP.

  • WHATWG is comprised of representatives of Apple (Safari), Mozilla (Firefox), Google (Chrome), Opera.
  • Driven by the emergence of web 2.0 and social websites (audio and video).

    FEATURES OF HTML5

    Semantic Tags

    Allows definition of html structure. Rather than using divs we now use semantic tags like header, footer, Nav, article, section.

    New Selector API

    Makes it easy to find elements in document rather than using the getelementbyId.

    New types of form input controls

    Like date picker, time picker, URL and email.

    Web Storage and web Sql
    allows storage on browser.

    Communication API.
    Geolocation i.e. maps.

    Web workers.

    Web sockets. Offline Working.

    Accessibility features.

    Inserting audio and videos without plugins.
    Graphics (canvas and SVG) Scalable Vector Graphics.
