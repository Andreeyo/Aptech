BORDER STYLE PROPERTY

Borders are rectangular outlines that surround an element. CSS border properties specifies the style, color, and width of the border. The border-style property applies a specific style to all four borders. Border style properties are: Border-bottom-style, Border-left-style, Border-right-style, Border-top-style.

Border Color Property

Applies colors to all four borders. Border color properties are:

  • Border-bottom-color
  • Border-left-color
  • Border-right-color
  • Border-top-color.

    • Border Width Property

    The border-width property is a shorthand property that specifies the width for all the four borders.

  • Border-bottom-width
  • Border-left-width
  • Border-right-width
  • Border-top-width.

    • Border Property

    Border property in CSS specifies all the properties such as style, width and color for all the four borders. The different border property include:

  • Border-bottom
  • Border-top
  • Border-left
  • Border-right
