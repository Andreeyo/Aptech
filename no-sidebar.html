TABLES

Tables allow user to view his/her data in a structured and classified format. Tables can contain any type of data such as text, images, links and other tables. A table is made up of rows and columns. The intersection of each row and column is called a cell. A row is made of sets of cells that are placed horizontally. A column is made up of a set of cells that are placed vertically. The TR element divides the table into rows and the TD element specifies columns for each row. By default a table has no border. The border attribute of the Table element specifies a border for making the table visible in a webpage.

TABLE HEADINGS

You can specify the heading for each column in HTML. To do this you need the TH element. The text in between the TH element appears in Bold.

COLSPAN ATTRIBUTE

Spanning refers to a process of extending a cell across multiple rows or columns. To span two or more columns, you use the colspan attribute of the TD and TH elements. The colspan attribute allows you to span cell along a horizontal row. The value of the colspan attribute specifies the number of cells across which a specific cell shall be expanded.

ROWSPAN ATTRIBUTE

The rowspan attribute spans a data cell across two or more rows. It allows you to span a data along a vertical column.
