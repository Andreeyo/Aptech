MARGIN PROPERTY

Margin is defined as the white space surrounding the content within an element. The different margin properties are:Margin-bottom, Margin-left,Margin-right,Margin-top.

PADDING PROPERTIES

Padding is a white space between the border of an element and its content. Padding property cannot accept negative values. The different padding properties are: Padding-bottom Padding-left, Padding-right, Padding-top.

POSITIONING

In CSS there are three types of positioning, namely normal, absolute and relative. The position property in CSS specifies the position of the content. By default, the value of the position property is static, which means the content is displayed in a normal flow. This means, the contents of the web page are placed in a sequence that is specified by the HTML source.

ABSOLUTE POSITIONING

Changes the position of elements content based on the specified co-ordinates. It removes the content from the original flow of the content. The element with absolute positioning does not affect the other elements in a flow. To specify that the content within an element should be absolutely positioned, you set the value of the position property to absolute.

RELATIVE POSITIONING

Relative positioning places the content within an element by considering the normal flow of the document. This means, it displaces the content to a new position without making any impact on the normal flow. It moves the content horizontally and/or vertically with respect to its original flow. To specify set the position property to relative.

POSITION PROPERTIES

The content with an element has four sides namely, left, right, top and bottom. These four sides makes up an elements box to which you can apply the various CSS position properties. The position properties specify the position of the elements content along the x and y axis.


PROPERTY DESCRIPTION
bottom Specifies the distance between the bottom end of an element and the top end of the parent element.
left Specifies the distance between the left end of an element and the left end of the parent element.
right Specifies the distance between the right end of an element and the right end of the parent element.
top Specifies the distance between the top end of an element and the top end of the parent element.

DISPLAY PROPERTY

The display property specifies the way the content is displayed on the web page. It specifies whether the content would contain the line-breaks or not.


PROPERTY DESCRIPTION
none Specifies that the content would not be displayed.
block Specifies that the content would be displayed with a line break before or after it.
inline Specifies that the content would be dis-played without any line break before or after it.

PSEUDO CLASSES

Pseudo classes allow you to apply different styles to your elements such as buttons hyperlinks etc.
Selector_name:state_name{ property:value}


STATE DESCRIPTION
active Defines a different style to an element that is activated by the user.
hover Defines a different style to an element when the mouse pointer is moved over it.
link Defines a different style to an unvisited hyperlink.
visited Defines a different style to the visited hyperlink.
