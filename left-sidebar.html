STYLE ATTRIBUTE

A style is a way to specify the visual appearance of the content. Web pages displays contents in different styles such as in different colors, fonts, sizes, and borders. This is done using the style attribute, which makes the web page look more attractive and pleasing. This attribute is used with most html elements and is specified within their start tags. Such a manner of specifying styles is called Inline CSS.
NOTE: The style attribute cannot be used with the HTML, HEAD, META and TITLE elements.

PROPERTIES OF STYLE ATTRIBUTE

The style attribute defines various properties, which specifies the style to be applied to the content. The different styles that you can specify include color, font, alignment and size.

Examples are font-family, font-weight, font-size, font-align, font-style.

HYPERLINKS

A website is a collection of web pages, which facilitate sharing information on the internet. The A (anchor) element is used to create a hyperlink. You can specify a text or an image as a hyperlink. To specify the linked page section or linked web page, you must use the attributes of the A element. The A element cannot contain any other A elements

COLOR PROPERTY

The color property of the style attribute sets the color of the text or background in a web page. In HTML, colors can be specified by using the color name, hexadecimal value, or RGB value. A RGB value is a combination of red, green and blue (RGB) colors contained within a color. Each color value can range from 0 to 255.

A hexadecimal value consists of six alphanumeric characters, which is a combination of 0-9 digits and A-F letters preceded with a hash symbol. It is another way to represent the RGB combination. The first two characters represents the amount of red color, while the next two represents that of green. Finally, the last two represents amount of blue color. The 0(hex #00) indicates no amount of a color and 255 (hex #FF) indicates 100% of a color. The table lists the commonly used colors in HTML.

COLOR NAME HEXADECIMAL CODE RGB VALUE
Black #000000 rgb (0,0,0)
Blue #0000FF rgb (0,0,255)
Green #00FF00 rgb (0,255,0)
Red #FF0000 rgb (255,0,0)
White #FFFFFF rgb (255,255,255)
Yellow #FFF000 rgb (255,255,0)

PHRASE ELEMENTS

Are container elements that add structural information to the text in a web page. Using the phrase elements, you can specify a full form of an acronym when the user moves the mouse over it.

ELEMENT DESCRIPTION
ACRONYM Displays an acronym in a webpage.
When the user hovers on the acronym,
its full form appears in a small pop-up as a tool tip.
CITE Specifies citations, such as titles of the newspapers
and references to other sources.
Such text is displayed in italic format in the browser.
