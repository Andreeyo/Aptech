DATA TYPES

JavaScript is referred to as the loosely typed language. This means that a variable holding a number can also hold a string later. Data types in JS are classified into two broad categories namely: primitive and composite data types. Primitive data types contain only a single literal value such as number or a string a literal is a static value, whereas the composite data types contain a group of values.

DECLARING VARIABLES

PRIMITIVE DATA TYPE DESCRIPTION
Boolean Contains only two values namely, true or false.
Null Contains only one value namely, null. A variable of this value specifies that the variables has no value. This null value is a keyword and it is not the same as the value, zero.
Number Contains positive and negative numbers and numbers with decimal point. Some of the valid examples include 6, 7.5, -8, 7.5e-3 etc.
String Contains alphanumeric characters in single or double quotation marks. The single quotes is used to represent a string, which itself consists of quotations. A set of quotes without any characters within it is known as the null string.

In JS, all composite data types are treated as objects. A composite data type can either be predefined or user-defined.

COMPOSITE DATA TYPE DESCRIPTION
Objects Refers to a collection of properties and functions. Properties specify the characteristics and functions determine the behavior of a JS object.
Functions Refers to a collection of statements, which are instructions to achieve a specific task.
Arrays Refers to a collection of values stored in adjacent memory locations.
